Semaine 3

4 février 2016

Cette semaine, nous avons principalement travaillé sur les quatre laboratoires sur Packet Tracer. Nous avons également mis en ligne le blogue, sur lequel sont affichés les journaux de bord pour chaque semaine.

Nous avons commencé à expérimenter avec l’API de Shodan.io en Python 3. Nous avons déjà un (dépôt de code sur GitHub) contenant une architecture de base pour démarrer notre projet de recherche. L’avancement du projet reste encore assez stagnant, mais nous prévoyons travailler cette fin de semaine afin d’accélérer la cadence.
